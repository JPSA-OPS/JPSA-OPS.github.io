# A's Fighting Game — Phaser Multi-page Prototype

This project is a browser-playable fighting game prototype (Phaser 3) with multiple pages:
- index.html — Main menu
- character-select.html — Choose fighter
- stage-select.html — Choose stage
- fight.html — Playable match (1vCPU)
- training.html — Training mode
- story.html — Story mode launcher
- roster.html — Full roster display

How to use
1. Put your music file at assets/music/theme.mp3 (supported mp3/ogg).
2. (Optional) Add sfx files under assets/sfx/.
3. Replace character sprites in assets/characters/ and stage backgrounds in assets/stages/.
4. Push this folder to a GitHub repository and enable GitHub Pages (branch: main, folder: root).
